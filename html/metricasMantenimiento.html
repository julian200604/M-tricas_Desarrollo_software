Métricas de mantenimiento


El estándar IEEE 982.1-1988 propone el Índice de Madurez del Software (IMS) como una métrica diseñada para evaluar la estabilidad de un producto de software, específicamente en el contexto de actividades de mantenimiento. Se calcula utilizando la siguiente fórmula:

IMS = [Mr - (Fa + Fc + Fd)] / Mr

Donde:

- Mr es el número de módulos en la versión actual.
- Fa es el número de módulos en la versión actual que se han añadido.
- Fc es el número de módulos en la versión actual que se han cambiado.
- Fd es el número de módulos de la versión anterior que se han borrado en la versión actual.

A medida que el IMS se acerca a 1.0, el producto de software comienza a estabilizarse. Esta métrica puede ser utilizada para la planificación de actividades de mantenimiento del software. Además, el tiempo medio para producir una versión de un producto de software puede correlacionarse con el IMS, lo que permite desarrollar modelos empíricos para el mantenimiento.
